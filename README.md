# script

Scripts to efficiently manage your infrastructure

# [Linux_mount_Check](https://github.com/lkimasu/script/tree/main/Linux_mount_Check)

Using [plink.exe](https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html) in Windows Powershell to check the mount status of the server

# [Windows_Remote_Mount_Check](https://github.com/lkimasu/script/tree/main/Windows_Remote_Mount_Check)

Using the [Invoke-command](https://learn.microsoft.com/ko-kr/powershell/module/microsoft.powershell.core/invoke-command?view=powershell-7.3) command in Windows PowerShell, remotely connect to the Windows server and check the mount status

# [Dns_Change_Window](https://github.com/lkimasu/script/tree/main/dns_change_window)

Change DNS server IP using CMD, PowerShell in Windows 2008, 2012 or later environment

# [window_Mount_Check(python)](https://github.com/lkimasu/script/tree/master/window_Mount_Check(python))

Check cpu, memory, disk utilization and capacity of each drive (local, network) using Python script in Windows environment
