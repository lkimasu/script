# 스크립트

인프라를 효율적으로 관리하기 위한 스크립트 입니다.

# [Linux_mount_Check](https://github.com/lkimasu/script/tree/main/Linux_mount_Check)
---
[plink.exe](https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html) 를 사용하여 Windows Powershell에서 서버의 마운트 상태 확인 

# [Windows_Remote_Mount_Check](https://github.com/lkimasu/script/tree/main/Windows_Remote_Mount_Check)

Windows PowerShell에서 Invoke-command 명령을 사용하여 Windows 서버에 원격으로 연결하고 마운트 상태 확인

# [Dns_Change_Window](https://github.com/lkimasu/script/tree/main/dns_change_window)

Windows 2008, 2012 이상 환경에서 CMD, PowerShell을 이용하여 DNS 서버 IP 변경

# [window_Mount_Check(python)](https://github.com/lkimasu/script/tree/master/window_Mount_Check(python))

Windows 환경에서 Python 스크립트를 사용하여 각 드라이브(로컬, 네트워크)의 CPU, 메모리, 디스크 사용률 및 용량 확인
